# TodoSpringCore
Todo application using Spring Core framework with CRUD functionality

## Prerequisites
- Java JDK 17 or later
- Maven 3.6+
- MySQL Database
- IDE (IntelliJ IDEA, Eclipse, or VS Code)

## Generate Maven Project Structure

### Option 1: Using Maven Archetype (Automatic)

Navigate to your project directory and run:

```powershell
cd C:\Users\mks94\Documents\Projects\TodoSpringCore
mvn archetype:generate -DgroupId=com.todo -DartifactId=TodoSpringCore -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
```

This automatically creates:
- `src/main/java` - Java source code
- `src/test/java` - Test code
- `pom.xml` - Maven configuration file

**Note:** This creates a subfolder. Move contents up one level if needed.

### Option 2: Manual Folder Creation

Create the standard Maven directory structure manually:

```powershell
# Navigate to project directory
cd C:\Users\mks94\Documents\Projects\TodoSpringCore

# Create main source folders
New-Item -ItemType Directory -Path "src\main\java\com\todo\model" -Force
New-Item -ItemType Directory -Path "src\main\java\com\todo\dao" -Force
New-Item -ItemType Directory -Path "src\main\java\com\todo\service" -Force
New-Item -ItemType Directory -Path "src\main\resources" -Force

# Create test folders
New-Item -ItemType Directory -Path "src\test\java\com\todo" -Force
New-Item -ItemType Directory -Path "src\test\resources" -Force
```

## Project Setup

### 1. Database Setup

Create MySQL database:

```sql
CREATE DATABASE tododb;

USE tododb;

CREATE TABLE todos (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    description TEXT,
    completed BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 2. Build Project

```powershell
mvn clean compile
```

### 3. Run Application

```powershell
mvn exec:java -Dexec.mainClass="com.todo.Main"
```

## Technologies Used

- **Spring Core** - Dependency Injection and IoC Container
- **Spring JDBC** - Database operations
- **MySQL** - Database
- **Maven** - Build and dependency management
